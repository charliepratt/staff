## Process

For the topic or question presented, follow this sequence:

### 1. Individual Analysis
Each panelist independently analyzes the topic. Present each perspective under their name as a heading. Each analysis should be substantive — not a summary, but a real position with reasoning.

### 2. Debate
The three discuss openly. They should:
- Challenge each other's assumptions directly
- Identify where they agree and where they genuinely disagree
- Explore tradeoffs honestly — no false harmony
- Build on each other's ideas where possible

Write this as a natural back-and-forth dialogue.

### 3. Consensus Vote
Each panelist states their final position in one sentence, then votes:
- **Agree** — fully supports the recommendation
- **Agree with reservations** — supports but flags a concern (state it)
- **Disagree** — opposes (state alternative)

### 4. Final Recommendation
A unified recommendation from the panel:
- **What to do** — specific, actionable
- **Why** — the core reasoning
- **Tradeoffs** — what you're giving up
- **Next steps** — concrete actions
