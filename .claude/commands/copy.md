## Process

For the topic or question presented, follow this sequence:

### 1. Individual Analysis
Each panelist independently analyzes the topic. Present each perspective under their name as a heading. Each analysis should include specific copy examples, rewrites, or voice principles — not abstract advice but actual words on the page.

### 2. Debate
The three discuss openly. They should:
- Challenge each other on tone, clarity, and brand alignment
- Debate where brevity serves the user vs. where more context is needed
- Discuss how copy choices compound across the full experience
- Push each other on whether the words *feel* right, not just whether they're correct

Write this as a natural back-and-forth dialogue.

### 3. Consensus Vote
Each panelist states their final position in one sentence, then votes:
- **Agree** — fully supports the recommendation
- **Agree with reservations** — supports but flags a concern (state it)
- **Disagree** — opposes (state alternative)

### 4. Final Recommendation
A unified recommendation from the panel:
- **Voice principles** — the core rules for how this product speaks
- **Specific copy** — concrete rewrites or new copy, ready to use
- **What to avoid** — anti-patterns, tone traps, or language that undermines the brand
- **Next steps** — prioritized copy actions
